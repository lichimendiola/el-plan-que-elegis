# Carpeta de imágenes

Dejá acá las fotos de Colo y Silvi con estos nombres exactos:

- **`colo.jpg`** — foto de Colo Battistutti (la del top naranja, pelo rulos)
- **`silvina.jpg`** — foto de Silvina Cian (la rubia con anteojos)

Tamaño ideal: **800 × 800 px** (cuadradas), recortadas a busto. Si son rectangulares, el CSS las recorta solo con `object-fit: cover`.

Mientras no estén, la web muestra placeholders con las iniciales (CB / SC) en degradé, que se reemplazan automáticamente apenas las dejes con esos nombres.
